鍵盤門禁系統

介紹

Arduino是一種開放源代碼的電子開發平台，可用於創建各種電子裝置和原型。鍵盤門禁系統是一種安全控制系統，它使用密碼或代碼來控制門的進入。 將Arduino與鍵盤和門禁系統相結合，可以創建一個簡單的鍵盤門禁系統。在這個系統中，鍵盤用於輸入密碼，而Arduino則用於驅動門禁系統以控制門的進入。 在這個系統中，Arduino通過讀取鍵盤上的輸入字符來獲取密碼。然後，它比較輸入的密碼與預先設置的密碼是否匹配。如果匹配，Arduino就會向門禁系統發送信號，使門打開。如果密碼不匹配，門禁系統將保持關閉狀態。 鍵盤門禁系統可以用於許多場合，例如辦公室、學校、醫院、銀行等場所。它可以提高安全性，並且比傳統的鑰匙和門禁卡更加方便和易於管理。


這是一個基於 Arduino 的鍵盤門禁系統程式碼。使用者需要輸入正確的密碼（在此程式中預設為 "789"）才能解鎖。以下是程式碼的詳細解釋： 1. **包含函式庫**：程式碼首先包含了 和 兩個庫。 是控制伺服馬達的庫， 是讀取鍵盤輸入的庫。 2. **變數和常數的定義**：程式碼定義了一些必要的變數和常數。例如：密碼、伺服馬達的位置、鍵盤的列數和行數（在這裡是 4x4 的鍵盤）、密碼的位置、鍵盤的配置、每一列和每一行的針腳。 3. **設定**：在 `setup()` 函數中，程式設定了兩個 LED 燈的輸出針腳（一個表示鎖定，另一個表示解鎖）並將伺服馬達連接到針腳 11。最後，將系統設定為鎖定狀態。 4. **主要循環**：在 `loop()` 函數中，程式碼讀取鍵盤的輸入。如果輸入的是 '*' 或 '#'，則將密碼位置重設為 0，並將系統設定為鎖定狀態。如果輸入的是當前密碼位置的字符，則將密碼位置增加 1。如果密碼位置達到 3（表示輸入了完整的密碼），則將系統設定為解鎖狀態。 5. **鎖定和解鎖的函數**：`LockedPosition()` 函數控制系統的鎖定和解鎖。如果系統被鎖定，則打開紅色 LED 燈，關閉綠色 LED 燈，並將伺服馬達設定為 0 度（或者你可以選擇其他你希望的鎖定位置）。如果系統被解鎖，則關閉紅色 LED 燈，打開綠色 LED 燈，並將伺服馬達設定為 90 度（或者你可以選擇其他你希望的解鎖位置）。 總的來說，這是一個相當基本的鍵盤門禁系統。其主要功能為當使用者輸入正確的密碼時，系統將從鎖定狀態轉為解鎖狀態，並通過改變 LED 燈的狀態和伺服馬達的位置來表示當前的狀態。

Code

#include 

const byte ROWS = 4; //Four rows
const byte COLS = 4; //Four columns
char keys[ROWS][COLS] = {
  {'1','2','3','A'},
  {'4','5','6','B'},
  {'7','8','9','C'},
  {'S','0','H','D'}
};
byte rowPins[ROWS] = {9, 8, 7, 6}; //connect to the row pinouts of the keypad
byte colPins[COLS] = {5, 4, 3, 2}; //connect to the column pinouts of the keypad

Keypad keypad = Keypad( makeKeymap(keys), rowPins, colPins, ROWS, COLS );

String v_passcode="";

void setup(){
  Serial.begin(9600);
}

void loop(){
  char key = keypad.getKey();
  
  if (key != NO_KEY){
    //Serial.println(key);
    
    v_passcode = v_passcode + key;
    
    if(key=='A')
    {
    	Serial.println("Enter Password");
        v_passcode="";
    }
    
    if(key=='D')
    {
    	Serial.println("Validate the Password");
      	//Serial.println(v_passcode);
      	
      	if (v_passcode=="589D")
        {
          Serial.println("Access Granted");
        }
      	else
        {
        	Serial.println("Access Denied");
        }      
    }        
  }
}
